## üöÄ Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/parthsali/dev_tools.git
   cd dev_tools
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
