Fast, privacy-focused developer tools — everything runs in your browser, nothing is sent to a server. Live at **[tools.parthsali.com](https://tools.parthsali.com)**.

## ⚡ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: Shadcn UI & Radix Primitives
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🛠️ Available Tools

**Text & Code**

- **Diff Checker**: Compare text side-by-side.
- **Regex Tester**: specific regex testing & debugging.
- **Markdown Preview**: Real-time rendering.
- **HTML Entities**: Escape/unescape characters.

**Converters & Formatters**

- **JSON ↔ YAML**: Instant format conversion.
- **CSV ↔ JSON**: Spreadsheet data transformation.
- **Color Converter**: HEX, RGB, HSL translation.
- **JSON Formatter**: Validate and prettify data.

**Encoders & Security**

- **Base64**: Encode and decode content.
- **JWT Decoder**: Inspect tokens client-side.
- **Hash Generator**: MD5, SHA-1, SHA-256 partials.

**Generators & Utils**

- **UUID Generator**: Create standard v4 UUIDs.
- **Data Generator**: Mock data for testing.
- **HTTP Status Codes**: Quick reference guide.

## 🚀 Getting Started

1. **Clone the repository**

   ```bash
   git clone https://github.com/parthsali/dev_tools.git
   cd dev_tools
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
